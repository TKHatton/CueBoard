The Problem

Meetings run on software.
But software is fighting you.

Every virtual meeting has an invisible operator — someone hunting through nested menus for the breakout room button, the record toggle, the host transfer option. All while participants watch and wait.

The 8-Minute Rule
50% drop

If action items aren't captured and distributed within 8 minutes of a meeting ending, completion rates drop by half. After 24 hours, they drop by 80%. Nobody is tracking the key moments while they're happening — because there's no tool designed for it.

The Solution

Three pages. Every control you need.

CueBoard maps the MX Creative Console's LCD keypad to three distinct modes of meeting operation — each one built for a different job.

Page 01

Live Meeting Controls

The essentials every participant and host needs, with live status indicators on every LCD button.

  • Mute/Unmute
  • Camera
  • Record
  • Share Screen
  • Reactions
  • Quick Add All
  • Switch Host
  • Chat
  • Raise Hand
Page 02

Event Operator Mode

Purpose-built for the person running the tech — breakout rooms, timers, and crowd control in one touch.

  • Breakout Rooms
  • Set Timer
  • Mute All
  • Spotlight
  • Whiteboard
  • Waiting Room
  • Lock Meeting
  • Announce
  • End Breakout
Page 03

Recording Intelligence

Flag moments in real time. Export a structured summary the instant the meeting ends.

  • Flag Moment
  • Action Item
  • Decision Made
  • Follow Up
  • Screenshot
  • Transcription
  • Captions
  • Pause/Resume
  • Export Flags
Analog Precision

The dial adapts to context.

The MX Creative Dial and Actions Ring provide smooth, context-sensitive analog control that buttons alone can't deliver.

Volume Control

Default mode. Smooth analog adjustment of system or meeting audio.

Timer Duration

Press Timer, then twist. Set countdown duration with precise physical control.

Background Blur

Real-time blur intensity adjustment. Find the exact level between sharp and fully blurred.

Actions Ring

Quick-switch between all three keypad pages and access frequent shortcuts from the MX Master 4.

Platform Agnostic

One plugin. Every platform.
Same muscle memory.

CueBoard auto-detects your active meeting app and adapts all controls automatically. Switch platforms without relearning anything.
